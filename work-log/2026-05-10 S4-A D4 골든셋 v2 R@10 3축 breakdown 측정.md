## 4. 핵심 finding (정직히 명시)

### 4.1 caption gap 가설 반전

D4 진입 전 가설: caption_dependent=true 가 baseline 에서 손해 → D5 prompt v2 reingest 시 +X pp gain.

D4 측정: caption=true 가 baseline 에서 **R@10 +6.98pp 우위**. 가설 반대.

해석 후보:
1. caption=true 표본 18건 = vision_diagram(3) + table_lookup(6) + exact_fact|true(9) 의 합 → 본질적으로 정답이 명확한 row 이라 baseline 에서도 강함
2. table 본문 chunk 자체가 dense+lex 으로 매칭됨 → caption 보강 없이도 정답 retrieve
3. 표본 18건 = 통계적 유의성 부족, 잡음 가능

→ **D5 reingest 의 ROI 낮을 가능성 시그널** (단 결정 보류 — D5 phase 1 sample-report 의 신규 v2 chunk 1.5% 만으로는 판정 불가).

### 4.2 cross_doc 폭락

cross_doc R@10 = **0.1667** (overall −0.5360pp). 표본 4건이지만 일관된 약점.

→ 후보 #3 **S4-A D3 Phase 2 cross_doc 보강** (4 → 8~10 row 확장) 의 우선순위 격상 신호.

### 4.3 doc 매칭 fail 11건 → 단일 doc 집중

- 10/11건 = 포트폴리오 PDF (chunk page DB NULL) — D4 와 무관한 잔존 한계 (핸드오프 §5 #6)

→ **D4 자체의 회귀 0**. 11건은 별개 이슈로 분류.
