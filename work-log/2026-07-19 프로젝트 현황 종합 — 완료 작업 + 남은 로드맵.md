## 1. 프로젝트 한 줄

**Jet-Rag** — 한국 공공기관 문서(PDF·HWP·이미지)를 올려두면, 시간이 지나 어렴풋한 기억만 남았을 때 자연어로 검색해 그 자료(문서·페이지·원문 인용)를 찾아주는 개인 지식 검색 에이전트. **모바일 우선.**

- **Live**: https://jetrag.woong-s.com (프론트 Vercel) / https://jetrag-api.woong-s.com (백엔드 Railway) / DB·Auth·Storage Supabase
- **스택**: api = Python 3.12 FastAPI (uv) / web = Next.js 16 + React 19 + Tailwind v4 (pnpm) / 임베딩 DeepInfra BGE-M3 / Vision·LLM Gemini
- **운영 모드**: 익명 = owner 12 docs read-only 데모 / 로그인 = 본인 격리 공간 업로드·검색 (3중 격리: user_id 필터 + RLS + Storage prefix)
- **월 비용**: ~$6 (Railway $5 + DeepInfra <$1). Supabase 이전 검토는 비추천 결론(연 $60 절감 vs 22k줄 재작성 + HWP 파싱 Deno 부재) → **Railway 유지 확정**
