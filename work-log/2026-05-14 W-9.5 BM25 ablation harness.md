## 3. 사용법

KPI #7 "하이브리드 우세 +5pp" 측정 — 같은 golden_v2 를 3 mode 로 측정:

```bash
cd api && uv run python ../evals/run_s4_a_d4_breakdown_eval.py \
  --mode hybrid --out ../evals/results/m3_kpi7_hybrid_$(date +%Y%m%d).md \
  --out-json ../evals/results/m3_kpi7_hybrid_$(date +%Y%m%d).json

cd api && uv run python ../evals/run_s4_a_d4_breakdown_eval.py \
  --mode dense --out ../evals/results/m3_kpi7_dense_$(date +%Y%m%d).md \
  --out-json ../evals/results/m3_kpi7_dense_$(date +%Y%m%d).json

cd api && uv run python ../evals/run_s4_a_d4_breakdown_eval.py \
  --mode sparse --out ../evals/results/m3_kpi7_sparse_$(date +%Y%m%d).md \
  --out-json ../evals/results/m3_kpi7_sparse_$(date +%Y%m%d).json
```

→ 3개 결과 R@10/top-1 비교. 기대: hybrid > dense + 5pp / hybrid > sparse + 5pp.
