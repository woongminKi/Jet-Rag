## 0. Jet-Rag 가 뭔가요?

**한국어 문서를 자연어로 검색·답변하는 RAG 서비스**입니다.
- 사업보고서·법률판례·정책 문서·학술논문·이력서 등 12개 한국어 문서가 미리 인덱싱돼 있습니다
- "2026 한국 경제 성장률 전망" 같은 자연어 질문 → 문서 본문에서 근거를 찾아 답변

라이브 사이트: **<https://jetrag.woong-s.com>**

---

## 1. 잠깐, 앱스토어에 없는데 어떻게 폰에 설치하나요?

**좋은 질문입니다.** Jet-Rag 는 **PWA (Progressive Web App)** 라는 방식을 사용해 **App Store / Play Store 등록 없이 폰에 설치**됩니다.

비유하자면:
- 웹사이트 즐겨찾기를 홈 화면에 추가하는 것과 비슷합니다
- 단, 홈 화면에서 탭하면 일반 앱처럼 (주소창 없이) 열립니다
- 설치 / 제거 모두 1분 안에 가능, 흔적 0

따라서 다음 단계는 모두 **폰 브라우저** 에서 진행하면 됩니다.

## 2. 안드로이드 (Android) 폰 — 5분

### 준비물
- Android 폰 (대부분 정상 동작)
- **Chrome 앱** (기본 탑재돼 있습니다)

### Step 1 — 사이트 열기

1. 폰에서 **Chrome** 앱 실행
2. 주소창에 입력: **`jetrag.woong-s.com`** (https:// 안 붙여도 됩니다)
3. Jet-Rag 홈 화면 표시 확인

### Step 2 — 앱 설치

1. Chrome 우상단 **점 3개 (⋮)** 탭
2. 메뉴를 아래로 살펴보면 **"앱 설치"** 또는 **"홈 화면에 추가"** 가 보입니다
3. 탭 → 팝업 → **"설치"** 선택
4. 폰 홈 화면에 **Jet-Rag 아이콘** (검정 배경 + 흰색 마크) 생성 확인

### Step 3 — 실행

1. 홈 화면의 Jet-Rag 아이콘 탭
2. **주소창이 없는** 풀스크린으로 열림 → 진짜 앱 같죠?

이제 일반 앱처럼 검색·답변에 사용하시면 됩니다.

### (선택) Step 4 — PDF 공유 기능 테스트

Adobe Acrobat / Google Drive 등에서 PDF 를 Jet-Rag 로 직접 보낼 수 있습니다.

1. 폰에 PDF 가 있는 앱 (Drive·카톡·Acrobat 등) 에서 PDF 1개 열기
2. **공유 아이콘 (↗ 또는 ⋮ → 공유)** 탭
3. 공유 시트에서 **"Jet-Rag"** 항목 보이면 성공
4. 탭 → 현재는 데모 모드라 "현재 데모 모드입니다..." 메시지 표시 (정상 동작)
5. 멀티유저 모드 진입 후에는 본인 문서로 자동 업로드됩니다

## 3. 아이폰 (iOS) — 5분

### 준비물
- iPhone (iOS 16 이상 권장)
- **Safari 앱** (Chrome 으로는 PWA 설치 불가, **반드시 Safari 사용**)

### Step 1 — 사이트 열기

1. 아이폰에서 **Safari** 실행
2. 주소창에 **`jetrag.woong-s.com`**
3. Jet-Rag 홈 화면 표시 확인

### Step 2 — 홈 화면에 추가

1. Safari 하단 (또는 상단) 의 **공유 아이콘** (네모 + 위쪽 화살표 ↑) 탭
2. 메뉴를 아래로 스크롤 → **"홈 화면에 추가"** 탭
3. 미리보기 → 우상단 **"추가"** 탭
4. 폰 홈 화면에 Jet-Rag 아이콘 생성 확인

### Step 3 — 실행

1. 홈 화면 Jet-Rag 아이콘 탭
2. **주소창이 없는** 풀스크린으로 열림

### (선택) Step 4 — PDF 공유 기능 (단축어 1회 설정 필요)

iOS Safari 는 공유 시트 자동 등록을 지원하지 않습니다. 단축어 앱으로 우회:

→ 별도 가이드 참조: `work-log/2026-05-28 iOS Shortcuts PDF 공유 가이드.md`
- 1회 ~3분 설정
- 설정 후 Acrobat 등에서 PDF 공유 시 "Jet-Rag 로 보내기" 항목 자동 출현

**Step 4 는 건너뛰어도 됩니다** — 검색·답변 기능 검증에는 영향 없습니다.

## 4. 사용해보기 (5분)

### 추천 검색어 5종

설치 후 Jet-Rag 홈 화면 하단 **추천 칩** 을 탭하거나, 직접 입력해 결과를 확인해보세요:

| 검색어 | 무엇을 보여주나 |
|---|---|
| `2026 한국 경제 성장률 전망` | 한국 경제 보고서에서 성장률·물가 등 정량 데이터 답변 |
| `SK 사업보고서 매출 흐름` | SK 그룹 사업보고서 표 데이터 (Vision OCR) |
| `하도급 직접지급합의 묵시적 해지 요건` | 법률 판례 RAG (복잡한 한국어 + 인용) |
| `데이터센터 산업 활성화 지원 사업 내용` | 정책 문서 다중 단락 추론 |
| `기웅민 이력서 프로젝트 경험` | 이력서에서 본인 경험 어필 |

### 검색 결과 카드 보는 법

- **매칭 강도 %**: 질문과 문서 본문 일치도
- **snippet ±240자**: 답변 근거 본문
- **doc 단위 그룹화**: 같은 문서 내 여러 결과 묶음

### AI 답변 (RAG)

검색 결과 화면 상단 **"AI 답변"** 탭 → Gemini 2.5 Flash 가 본문 근거로 답변 생성. **출처 highlight** + **Ragas 신뢰도 점수** 표시.

## 5. 안 되는 경우 — 트러블슈팅

### Android — "앱 설치" 메뉴가 안 보여요

1. 사이트가 정상 로드됐는지 확인 (jetrag.woong-s.com 검정 홈 화면)
2. Chrome 메뉴 다시 열기 (스크롤 더 내려서 찾아보기)
3. 그래도 없으면 → Chrome 우상단 점 3개 → "홈 화면에 추가" 라도 가능
4. 그래도 안 되면 → Chrome 업데이트 (Play Store) 후 재시도

### iOS — "홈 화면에 추가" 가 안 보여요

1. **Safari** 인지 확인 (Chrome iOS 는 PWA 설치 미지원)
2. Safari 하단 가운데 공유 아이콘 (■↑) 확인
3. 메뉴 아래로 스크롤 (홈 화면에 추가는 보통 두 번째 줄)
4. 시크릿 모드 (Private Browsing) 가 아닌지 확인

### 검색 결과가 안 보여요

1. 인터넷 연결 확인
2. 위 추천 검색어 5종으로 시도 (한국어 문서만 인덱싱돼 있어 한국어 query 권장)
3. 그래도 안 되면 → 잠시 후 재시도 (백엔드 cold start ~10초 가능)

### 공유 시트에 Jet-Rag 가 안 보여요

1. PWA 설치 먼저 완료 확인 (홈 화면에 아이콘 있는지)
2. PDF 형식 인지 확인 (Jet-Rag 는 PDF 만 받습니다)
3. Android 만 자동 등록 (iOS 는 단축어 설정 필요)

## 7. 자주 묻는 질문

### Q1. 내 데이터가 어디로 가나요?
- 현재 데모 모드 (read-only) — 사용자 업로드 모두 차단 (503 응답). 본인 데이터 안 모입니다
- 멀티유저 모드 진입 후에는 본인 데이터가 본인 user_id 아래에 RLS 격리 저장됩니다

### Q2. 비용이 드나요?
- 베타 테스트 무료
- 백엔드 운영 비용 ~$5/월 (개인 부담)

### Q3. PC 에서도 쓸 수 있나요?
- 네, jetrag.woong-s.com 그대로 PC 브라우저에서 사용 가능
- PC Chrome 에서도 "앱 설치" 가능

### Q4. PWA 가 제 폰에 영향 주나요?
- 일반 즐겨찾기와 거의 동일 — 사이트 데이터만 소량 캐시
- 제거는 다른 앱처럼 길게 누름 → 삭제 (흔적 0)

### Q5. 베타가 끝나면 어떻게 되나요?
- 데모 모드 유지 — 채용 담당자가 본 사이트를 이력서 포트폴리오로 계속 볼 수 있습니다
- 본인 업로드 기능은 멀티유저 D1 Auth 진입 후 본격 ON
