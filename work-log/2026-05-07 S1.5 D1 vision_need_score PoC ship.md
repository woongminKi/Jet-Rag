### 2.2 휴리스틱 (master plan §6 S1.5 default)

```python
# (a) entity regex — case-insensitive
_ENTITY_PATTERN = r"(\[표\s*\d+\]?|\[그림\s*\d+\]?|<표\s*\d+>|<그림\s*\d+>"
                  r"|Figure\s*\d+|Table\s*\d+|식\s*\(\s*\d+\s*\)|Eq\.\s*\(\s*\d+\s*\))"
# (b) table-like — 한 line 내 ≥3 span 비율, ≥0.5 면 신호
_TABLE_MIN_SPANS_PER_LINE = 3
_TABLE_LIKE_THRESHOLD = 0.5
# (c) text_density — chars / pt², < 1e-3 이면 sparse 신호 (A4 ~500 chars 이하)
_DENSITY_THRESHOLD = 1e-3
```

`needs_vision = (a)>0 OR (b)≥0.5 OR (c)<1e-3` — OR 합산. 가중·임계 조정은 **D2** 분포 측정 후.
