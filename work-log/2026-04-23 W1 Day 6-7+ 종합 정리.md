| reingest 동시성 | latest job status 체크로 409 Conflict | 단일 사용자 MVP 충분 |
| embed 스테이지 적재 | row 단위 update (upsert 폐기) | NOT NULL doc_id violation 회피 + 의미 명확 |
| 의존성 추가 | API 0개 / Web shadcn 7개만 | CLAUDE.md "의존성 추가 사전 승인" |
