## §13 hydration fix (사후 보강 ship)

### 증상

브라우저 console 에 hydration mismatch 발생:
```
Hydration failed because the server rendered text didn't match the client.
- A SSR-ed Client Component used: ...
+ data-state="closed" data-slot="tooltip-trigger"
- data-state={null} data-slot={null}
+ 매칭 강도
- 100
```

server 가 단순 `<span>100</span>` 을 그렸는데 client hydration 후 `<button data-state="closed">매칭 강도</button>` 로 트리 자체가 달라짐.

### 원인

- `web/src/components/jet-rag/result-card.tsx` 가 `'use client'` 미선언 (RSC)
